# Міграція вручну

1. Відкрий [Supabase SQL Editor](https://supabase.com/dashboard/project/jpktswxqeanamkymqwjg/sql/new)
2. Скопіюй весь вміст з `migrations/20250214000000_init.sql`
3. Вставь у редактор і натисни **Run** (або Ctrl+Enter)

Після цього додай тестові сорти (опційно):
- Відкрий той самий SQL Editor
- Скопіюй вміст з `seed.sql`
- Run
