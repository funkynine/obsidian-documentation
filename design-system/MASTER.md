# Design System: Тюльпани Вінниця

## Product Type
Flower e-commerce, seasonal (14 лютого, 8 березня), local delivery (Вінниця, Жмеринка, Бар).

## Color Palette

| Token | Hex | Usage |
|-------|-----|-------|
| Primary | #B91C1C (rose-700) | CTAs, links, accents |
| Primary Hover | #991B1B (rose-800) | CTA hover |
| Secondary | #059669 (emerald-600) | Success, freshness, nature |
| Background | #FFFBEB (amber-50) | Warm cream, romantic |
| Background Alt | #FFFFFF | Cards, sections |
| Text | #1C1917 (stone-900) | Headings, body |
| Text Muted | #57534E (stone-600) | Descriptions |
| Border | #E7E5E4 (stone-200) | Dividers, cards |
| Accent | #D97706 (amber-600) | Highlights, urgency |

## Typography
- **Headings:** Playfair Display (elegant, romantic serif) or Cormorant Garamond
- **Body:** Nunito or Source Sans 3 (friendly, readable)
- **Scale:** Hero 3rem/48px, H2 2rem/32px, body 1rem/16px

## Style
- **Aesthetic:** Romantic, warm, elegant, trustworthy
- **Effects:** Subtle card shadows, soft rounded corners (rounded-xl), hover lift on cards
- **Icons:** Lucide (already in shadcn)

## Landing Structure
1. Hero: Headline + subheadline + CTA "Замовити тюльпани"
2. Catalog: Grid of variety cards (image, name, price, "Обрати")
3. Occasion blocks: "14 лютого" / "8 березня" with CTA
4. Footer: Cities, contacts

## Conversion
- CTA: High contrast (rose on cream), rounded-full for pill shape
- Sticky navbar with CTA on scroll
